# MONEY BIZ PRD

## 1. 제품 한 줄 정의

`1인 사업자와 소상공인을 위해 매출을 높이는 다음 행동을 추천하고 바로 실행하게 만드는 AI 코치 앱`

## 2. 제품 목표

- 사용자가 매일 무엇을 해야 매출에 도움이 되는지 바로 판단하게 한다.
- 마케팅, 가격, 재구매 유도, 고객 후속 관리의 실행 시간을 줄인다.
- 사용자가 앱을 `분석 툴`이 아니라 `매출 실행 툴`로 인식하게 만든다.

## 3. 비목표

- 회계 ERP 대체
- 세무 신고 자동화
- 대기업용 CRM
- 광고 플랫폼 완전 자동 운영

## 4. 핵심 사용자

### 사용자 A: 1인 온라인 셀러

- 네이버 스마트스토어, 인스타그램, 오픈마켓 등에서 판매한다.
- 고객 응대, 가격 조정, 홍보 문구 작성까지 혼자 처리한다.
- 무엇부터 손대야 매출이 오를지 판단이 어렵다.

### 사용자 B: 예약형 소상공인

- 미용실, 필라테스, 공방, 레슨형 사업자
- 재방문 유도 메시지와 프로모션 기획이 약하다.
- 빈 시간대 채우기와 단골 전환이 중요하다.

### 사용자 C: 소규모 팀 사업자

- 2~5명 팀으로 운영한다.
- 간단한 승인과 역할 분담이 필요하다.
- 매장별 성과 비교와 공통 템플릿 관리 니즈가 있다.

## 5. JTBD

- 오늘 무엇을 해야 매출에 도움이 되는지 빠르게 알고 싶다.
- 홍보 문구나 메시지를 고민하는 시간을 줄이고 싶다.
- 내가 한 액션이 실제 매출에 도움이 되었는지 확인하고 싶다.
- 가격을 낮추지 않고도 수익을 올릴 방법을 찾고 싶다.

## 6. 핵심 사용자 흐름

1. 회원가입
2. 업종/채널/목표 매출 입력
3. 상품/서비스 정보와 평균 가격 입력
4. 최근 매출/비용 데이터 입력 또는 CSV 업로드
5. 앱이 오늘의 매출 액션 3개 제안
6. 사용자가 액션을 실행하거나 수정
7. 실행 결과를 기록
8. 앱이 다음 액션을 재추천

## 7. MVP 기능 요구사항

### P0

- 이메일/소셜 로그인
- 업종 기반 온보딩
- 상품/서비스 등록
- 최근 매출/비용 수기 입력
- CSV 업로드
- 오늘의 액션 피드
- AI 카피 생성
- 마진 계산기
- 액션 실행 체크
- 기본 KPI 대시보드

### P1

- 고객 세그먼트별 메시지 추천
- 재구매/휴면 고객 리마인드 템플릿
- 업종별 템플릿 라이브러리
- 다중 채널 성과 비교
- 팀원 초대

### P2

- POS/예약 시스템 연동
- 광고 채널 연동
- 자동 캠페인 생성
- AI 에이전트형 주간 계획

## 8. 상세 기능 정의

### 8.1 온보딩

- 입력값: 업종, 판매 채널, 월 목표 매출, 평균 객단가, 주요 고객층
- 결과물: 개인화된 홈 화면과 추천 초기 설정

### 8.2 오늘의 액션 피드

- 홈에서 가장 먼저 노출
- 카드 3장 구성
- 각 카드에는 `이유`, `예상 효과`, `실행 버튼`, `완료 체크`를 포함

예시:

- 가격을 5% 인상하되 세트 구성 추가
- 재구매 고객에게 7일 내 쿠폰 메시지 발송
- 인스타 릴스용 홍보 문구 3종 게시

### 8.3 AI 생성 도구

- 채널별 홍보 문구 생성
- DM/문자/알림톡용 리텐션 메시지 생성
- 상품 설명, 쿠폰 제목, 후기 요청 문구 생성

### 8.4 마진 계산기

- 입력값: 판매가, 원가, 수수료, 배달/배송비, 광고비, 쿠폰 비용
- 출력값: 순이익, 마진율, 손익분기 판매량, 권장 가격 범위

### 8.5 성과 추적

- 액션별 실행 여부 기록
- 액션 이후 매출/예약/문의 변화 기록
- 다음 추천에 반영

## 9. 핵심 화면

- 로그인 / 회원가입
- 온보딩
- 홈 대시보드
- 오늘의 액션 상세
- AI 생성 스튜디오
- 마진 계산기
- 성과 리포트
- 설정 / 결제

## 10. KPI

### 선행 KPI

- 가입 후 온보딩 완료율
- 첫 데이터 입력 완료율
- 7일 내 첫 액션 실행률
- 액션 카드 클릭률
- AI 생성 사용률

### 후행 KPI

- 무료에서 유료 전환율
- 4주 유지율
- 액션 실행 사용자군의 매출 개선율
- ARPPU
- Add-on 구매율

## 11. 성공 기준

MVP 출시 후 90일 기준 가정:

- 온보딩 완료율 60% 이상
- 첫 액션 실행률 35% 이상
- 무료 사용자 대비 유료 전환율 4% 이상
- 주간 유지율 30% 이상
- 유료 사용자 월 이탈률 8% 이하

## 12. 리스크와 대응

- 데이터 입력 피로도: 초기에는 수기 입력 항목을 최소화한다.
- AI 결과 품질 편차: 업종별 템플릿과 톤 가이드를 같이 제공한다.
- 과금 명분 부족: 리포트보다 실행 결과를 전면에 둔다.
- 복잡한 연동 요구: MVP는 CSV와 수기로 시작하고, 연동은 P2로 미룬다.

## 13. 출시 순서

### Phase 1

- 온라인 셀러 대상 MVP
- 수기 입력 + CSV
- 액션 추천 + AI 카피 + 마진 계산

### Phase 2

- 예약형 사업자 확장
- 재방문 캠페인 추천
- 업종별 템플릿 강화

### Phase 3

- 다중 매장/팀 기능
- 외부 시스템 연동
- 자동화 에이전트
