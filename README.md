# MONEY

수익화 가능한 앱 아이디어를 비교하고, 바로 검증 가능한 주력안을 선정한 초기 설계 문서다.

현재 결론은 `MONEY BIZ`를 1순위로 추천한다.

## 추천 결과

### 1. 주력안: MONEY BIZ

`1인 사업자 / 소상공인용 AI 매출 자동화 코치`

- 상품, 가격, 매출 목표, 채널 정보를 입력하면 오늘 해야 할 수익 행동을 추천한다.
- 추천 행동은 `가격 조정`, `홍보 문구 생성`, `재구매 메시지`, `캠페인 아이디어`, `고객 후속 액션`으로 구성한다.
- 추천만 끝내지 않고 바로 실행 가능한 템플릿과 체크리스트를 제공한다.

### 2. 차선안: MONEY CREATOR

`크리에이터용 수익원 통합 대시보드`

- 유튜브, 인스타, 클래스, 제휴, 디지털 상품 매출을 한 곳에서 관리한다.
- 장점은 구독형 과금이 가능하다는 점이다.
- 단점은 외부 플랫폼 연동 의존도가 크고 초기 데이터 연결 난도가 높다.

### 3. 차차선안: MONEY SELLER

`중고/오픈마켓 셀러용 마진·콘텐츠 자동화`

- 판매 수수료, 택배비, 광고비를 반영해 적정 가격과 마진을 계산한다.
- 상품 설명과 썸네일 문구를 자동 생성한다.
- 특정 셀러군에는 강하지만 타깃이 상대적으로 좁다.

## 왜 `MONEY BIZ`를 선택했는가

다음 자료를 종합한 추론이다.

- RevenueCat의 2025 구독 앱 리포트에서는 `Business` 카테고리가 높은 LTV를 보였다.
- 같은 리포트에서 `Asia-Pacific`의 구독 LTV도 경쟁력이 높게 나타났다.
- Sensor Tower 자료에서는 생성형 AI 앱 매출과 다운로드가 2024년에 크게 증가했다.
- QuickBooks와 U.S. Chamber 자료에서는 소규모 사업자의 AI 활용이 빠르게 늘고 있고, 주된 사용처가 `마케팅`, `고객 응대`, `관리 업무`, `장부/운영`으로 나타났다.

즉, `소비자가 돈을 버는 앱`보다 `사업자가 매출을 올리기 위해 돈을 내는 앱`이 더 직접적이고 과금 명분이 강하다.

## MONEY BIZ 핵심 설계

### 핵심 문제

- 1인 사업자와 소상공인은 `매출을 올릴 행동`을 매일 판단하기 어렵다.
- 마케팅, 가격 조정, 재구매 유도, 고객 응대가 분산되어 있다.
- 기존 툴은 회계나 CRM 한 축만 해결하고, `오늘 무엇을 해야 돈이 되는지`는 잘 제안하지 않는다.

### 핵심 가치 제안

- 데이터를 입력하면 `오늘 해야 할 3가지 수익 액션`을 제시한다.
- 액션마다 바로 사용할 수 있는 실행물까지 생성한다.
- 사용자는 분석보다 실행에 집중할 수 있다.

### MVP 범위

- 온보딩: 업종, 판매 채널, 가격대, 목표 매출, 고객 유형 입력
- 매출/비용 입력: 수기 입력 또는 CSV 업로드
- 일일 액션 피드: 오늘의 우선 행동 3개 추천
- AI 생성: 홍보 문구, 재구매 메시지, 쿠폰 문안, 상품 설명
- 마진 계산기: 가격/수수료/원가/광고비 기반 적정 마진 계산
- 성과 추적: 추천 액션 실행 여부와 결과 기록

### MVP에서 하지 않을 것

- 초반부터 복잡한 POS/ERP 직접 연동
- 정교한 세무 신고 자동화
- 대규모 팀 협업 기능
- 광고 자동 집행

## 수익 모델

- 무료: 액션 추천 제한, 저장 제한
- Pro: 월 구독, 일일 추천 확대, AI 생성 확대, 성과 분석 제공
- Team: 팀원 초대, 승인 플로우, 다중 매장
- Add-on: AI 크레딧, 업종별 템플릿 팩, 캠페인 팩

핵심은 `구독 + 일회성 add-on`의 하이브리드 모델이다.

## 권장 출시 전략

1. `모바일 우선 MVP`로 시작한다.
2. 초기에는 `네이버 스마트스토어 판매자`, `인스타 기반 소상공인`, `예약형 자영업` 중 하나로 세그먼트를 좁힌다.
3. 유입 메시지는 `AI`보다 `매출 액션 추천`에 둔다.
4. 첫 KPI는 다운로드보다 `7일 내 첫 액션 실행률`로 본다.
